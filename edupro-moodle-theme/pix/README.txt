Place the following image files in this directory:

  logo.png        — Edupro logo, white version, recommended 280x80px PNG (transparent background)
  favicon.ico     — 32x32 favicon
  og-image.png    — Open Graph / social share image, 1200x630px

These files are referenced in:
  layout/drawers.php    — $OUTPUT->image_url('logo', 'theme_edupro_boost')
  layout/login.php      — same
  lib.php               — og-image.png for SEO meta
